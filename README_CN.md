本项目纯粹就是瞎搞了一些新功能，提供了图形界面用于配置题目分布、权重、填空答案等，用于自动填写问卷星问卷并批量提交（苦逼大学生们有救了哈哈）

**主要特性**

- 💻具有友好的GUI界面，零门槛，无需编程基础，认得汉字就能用
- ⭐具备本地二维码解码功能：可直接上传问卷二维码图片自动识别出问卷链接
- 😄免费无限量，不再求助群聊朋友圈
- 🧑‍🤝‍🧑可以自动解析题目内容，并通过配置向导自己预设答案，调整选项概率和比例
- 🚀减少提交后的等待时间，提升刷问卷数量的速度
- 🌐模拟浏览器UA，破解仅微信可提交的问卷
- 📁支持保存问卷，方便多次复用同一份问卷的答案策略

**关键配置说明**

- `问卷链接`
  - 填写问卷星页面的完整链接（示例：`https://v.wjx.cn/vm/...`）。
  - 也可以直接点击「上传二维码」按钮，上传问卷二维码图片，程序会自动解码并填入链接。

- `目标份数`
  - 需要自动填写并提交的总份数。
  - 填得越大，执行时间越长；如果只是测试建议先填一个较小的数字（比如 3～5）。

- `浏览器数量`
  - 同时开启的浏览器实例数量（并发数）。
  - 数值越大，整体速度越快，但也越占用 CPU / 内存，可能会导致电脑变卡。
  - 一般家用电脑建议 2～5 之间，根据自身配置慢慢调。

- `分布方式`
  - `完全随机`：
    - 对于选择题，从所有候选选项中随机选取答案。
    - 适合对结果分布没有特殊要求，只想「快点填完」的场景。
  - `自定义权重`：
    - 可以为每个选项设置一个「权重」，程序会按权重比例随机选择答案。
    - 例如设置 A:50%、B:30%、C:20%，可以更贴近真实问卷数据分布。
    - 在 GUI 中可通过滑块或数值框直接调整权重，会实时显示占比。

- `题目列表区`
  - 左侧会展示当前问卷中解析出的所有题目及题型。
  - 单击某一题目可在右侧查看/编辑该题的答案预设与权重配置。
  - 对多选题、矩阵题等复杂题型，可以为每个子选项分别设置权重或固定选中状态。

- `全选` / 批量操作
  - 表头左上角的复选框为「全选」按钮：
    - 勾选：一次性选中所有题目；
    - 取消勾选：一次性取消所有题目的选中状态。
  - 在选中若干题目后，可以使用下方或右键菜单的「批量删除」「批量编辑」等操作：
    - 批量删除：移除这些题目的自定义配置（不影响问卷原始题目）。
    - 批量编辑：对多个题目同时应用相同的分布方式或答案策略。

- `填空题 / 主观题配置`
  - 对于填空题，可以：
    - 设定一个固定答案（每份问卷都填同样的内容），例如姓名、学院等；
    - 或者填写一个「候选答案列表」，程序会在列表中随机选取一个进行填写。
  - 对于主观题（长文本），可以：
    - 配置一段固定文本；
    - 或者预置多段回答，由程序随机抽取，避免每一份答案完全一样。

- `自动配置问卷`
  - 点击「自动配置问卷」按钮后，程序会：
    - 访问问卷链接，解析页面结构；
    - 自动识别题目类型（单选、多选、矩阵、填空等）；
    - 为部分常见题型给出默认配置（例如为所有选项分配均匀权重），你可以在此基础上微调。
  - 若解析失败，可检查问卷链接是否正确、问卷是否需要登录或是否设置了访问限制。

- `开始执行`
  - 确认上述参数配置无误后，点击「开始执行」：
    - 程序会按照设定的「目标份数」「浏览器数量」和各题目的策略自动填写并提交问卷；
    - 界面通常会显示当前进度、成功提交数量以及错误日志。
  - 执行过程中如需紧急停止，可直接关闭程序或根据界面提供的「停止」按钮进行中断（如果已实现该按钮）。
