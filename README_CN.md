# fuck-wjx (问卷星速填)

[English](./README.md) | 简体中文

> 参考了 [Zemelee/wjx](https://github.com/Zemelee/wjx)，别忘了给大佬点个star⭐

## ✨ 主要特性

| 功能 | 说明 |
|------|------|
| 💻 零门槛界面 | 不用写代码，认字就能用 |
| 📷 二维码识别 | 上传问卷二维码 → 自动提取链接 |
| 🎯 智能配置 | 自动解析题目，设置权重和概率 |
| 🎭 全真模拟 | 模拟真人打字速度和鼠标移动 |
| 🌍 随机IP | 绕过IP限制和验证码 |
| ⏸️ 随时停止 | 优雅停止，不丢失进度 |
| 📁 保存复用 | 导出配置，下次直接用 |

## 🚀 快速开始

> [!TIP]
> **懒得折腾？** 直接去 [Release](https://github.com/hungryM0/fuck-wjx/releases/latest) 下载exe，双击运行，完事。

### 从源码运行

```bash
pip install -r requirements.txt
python fuck-wjx.py
```

**环境要求：** Windows 10/11 或 Linux，Python 3.8+

---

## 📖 使用方法

1. 粘贴问卷链接或上传二维码
2. 点击「自动配置问卷」解析题目
3. 通过配置向导调整答案权重
4. 设置目标份数和浏览器数量
5. 点击「开始执行」，坐等收工

---

## ⚙️ 关键配置

| 配置项 | 作用 |
|--------|------|
| 目标份数 | 要提交的总份数（建议先用3-5份测试） |
| 浏览器数量 | 并发数量（一般电脑建议2-5个） |
| 分布方式 | 完全随机或自定义权重 |
| 全真模拟 | 模拟真人行为，慢但安全 |
| 随机IP | 绕过IP限制（可能产生费用） |
