## Run & Build

Requirements: macOS 10.15+, Python 3.8+

```bash
# Install dependencies
pip install -r requirements.txt
pip install pyinstaller

# Run
python fuck-wjx.py

# Build as .app
chmod +x build_macos.sh
./build_macos.sh
```
