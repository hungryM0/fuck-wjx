# fuck-wjx (问卷星速填)

参考了   https://github.com/Zemelee/wjx   的仓库，请不要忘记给这位佬给出一个star⭐

本项目纯粹就是瞎搞了一些新功能，提供了图形界面用于配置题目分布、权重、填空答案等，用于自动填写问卷星问卷并批量提交（苦逼大学生们有救了哈哈）

**注意：此项目仅供学习与测试目的使用，请确保对目标问卷有自动提交的授权，避免滥用或违反他人服务条款，严禁污染他人的问卷数据**

**主要特性**

- 💻具有友好的GUI界面，零门槛，无需编程基础，认得汉字就能用
- ⭐具备本地二维码解码功能：可直接上传问卷二维码图片自动识别出问卷链接
- 😄永久免费无限量，不再求助群聊朋友圈
- 🧑‍🤝‍🧑可以自动解析题目内容，并通过配置向导自己预设答案，调整答案概率和比例
- 🚀减少提交后的等待时间，提升刷问卷数量的速度
- 🤖绕过人机验证，自动模拟智能点击

- Windows 10/11、 Linux 均可运行
- Python 3.8+

**Python 依赖**

- selenium
- numpy
- pyzbar
- Pillow
- requests
- packaging
- beautifulsoup4

如果下载的是源码，先使用 pip 安装依赖：

```pwsh
pip install -r requirements.txt
```

---

**如何运行**

```pwsh
python fuck-wjx.py
```
或直接双击用Python打开fuck-wjx.py

在打开的 GUI 中：
   - 填写问卷链接，或点击"上传二维码"按钮上传二维码图片自动识别链接
   - 点击"自动配置问卷"解析题目
   - 使用配置向导添加答案预设
   - 根据自身需求，决定是否再手动添加一些题目配置
   - 设置目标份数与浏览器数量，点击"开始执行"

---

**关键配置说明**

- `问卷链接`：填写问卷星页面的链接（示例：`https://v.wjx.cn/vm/...`）。
- `目标份数`：想要刷取问卷的总份数。
- `浏览器数量`：并发浏览器数量（⚠️越多越快，但可能会造成电脑卡顿）。
- `分布方式`：
  - `完全随机`：随机选择答案； 
  - `自定义权重`：手动输入或使用滑块设置权重比例；
- `全选`：题目列表左上角的复选框用于一次勾选/取消勾选所有题目，支持批量删除与编辑。
